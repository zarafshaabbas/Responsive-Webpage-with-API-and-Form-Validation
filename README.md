# 🌐 Interactive Responsive Webpage with API & Form Validation

## 📌 Project Overview

This project is a complete frontend web application built using **HTML, CSS, and JavaScript**. It combines responsive design, dynamic API integration, user interaction, and form validation into a single webpage.

---

## ✨ Features

* Clean and structured HTML layout
* External CSS for styling and responsiveness
* JavaScript for interactivity
* Button to change background color dynamically
* External profile link (LinkedIn)
* API integration to fetch GitHub profile data
* Contact form with input validation
* Dynamic error and success messages

---

## 🛠️ Technologies Used

* **HTML5** – webpage structure
* **CSS3** – styling and responsive design
* **JavaScript (ES6)** – interactivity, validation, and API handling
* **GitHub API** – for fetching user profile data

---

## 🚀 How to Run the Project

1. Create a folder on your system
2. Add the following files:

   * `index.html`
   * `styles.css`
   * `script.js`
3. Ensure all files are in the same directory
4. Open `index.html` in any web browser

## ⚡ Functionality

### 🔹 Background Color Change

* Click the **“Change Background Color”** button
* The background color changes randomly

### 🔹 External Profile Link

* Click **“Visit My Profile”**
* Opens LinkedIn profile in a new tab

### 🔹 GitHub API Integration

* Fetches and displays:

  * Name
  * Bio
  * Public repositories
  * Followers
* Displays dynamically on the webpage

### 🔹 Form Validation

* User must fill all fields:

  * Name
  * Email
  * Message
* Shows error messages for invalid input
* Displays success message on valid submission

## 📱 Responsive Design

The webpage adapts to different screen sizes:

* **Desktop** → Full layout
* **Tablet (≤768px)** → Adjusted layout
* **Mobile (≤480px)** → Optimized for smaller screens
